# HealthCheck - Machine Learning Healthcare Prediction System

## Overview

HealthCheck is a Flask-based healthcare prediction application that uses Machine Learning models to predict different diseases based on user-provided medical parameters.

The application provides multiple prediction services:

- Diabetes Prediction
- Breast Cancer Prediction
- Heart Disease Prediction
- Fetal Health Prediction
- Stroke Prediction
- Liver Disease Prediction
- Malaria Image Detection
